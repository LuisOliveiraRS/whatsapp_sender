# Envio Automático de Boas-Vindas pelo WhatsApp

Este é um aplicativo desktop simples que envia mensagens de boas-vindas via **Twilio WhatsApp API** usando uma lista de contatos em Excel.

## 🛠 Funcionalidades

- ✅ Interface gráfica (Tkinter)
- ✅ Carrega planilha `.xlsx` com nomes, telefones e links
- ✅ Envia mensagem personalizada via WhatsApp
- ✅ Marca status como "Enviado" ou "Erro"
- ✅ Salva planilha atualizada

## 🔐 Requisitos

1. Conta no [Twilio](https://www.twilio.com/ )
2. WhatsApp número ativo no sandbox do Twilio
3. Python 3 instalado
